## About Me

I'm a final-year **B.Sc. Artificial Intelligence student** at Kafr El-Sheikh University, specializing in **Data Engineering** with a strong foundation in Machine Learning. 

My journey began in ML engineering, but I discovered my passion for building the infrastructure that powers AI at scale. I design and implement **production-grade data pipelines** that transform raw data into analytics-ready datasets, enabling data-driven decisions and ML deployments.

**What I do:**
- Build scalable **ELT/ETL pipelines** using modern data stacks (Snowflake, dbt, Airflow, PySpark)
- Design **data warehouses** with dimensional modeling and performance optimization
- Integrate **ML models** into data workflows for real-time predictions
- Deploy solutions on **AWS** & **Microsoft Fabric**

I'm passionate about building reliable data infrastructure that turns raw data into actionable insights. Whether it's designing pipelines that process millions of records or integrating ML models for real-time predictions, I focus on creating systems that are scalable, maintainable, and production-ready.
